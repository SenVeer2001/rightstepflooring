## Troubleshooting

### Courses Not Showing
- Check search box (clear if not needed)
- Check category filter (set to "All Categories")
- Check status filter (set to "All Status")

### Status Not Changing
- Click dropdown properly
- Select different status
- Should update immediately

### Filters Not Working
- Check if typed correctly in search
- Filters are case-insensitive
- Combine filters for better results

### Mobile Layout Issues
- Zoom out slightly if text too large
- Scroll horizontally for table on smaller screens
- Mobile cards are full-width
